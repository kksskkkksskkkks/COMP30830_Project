# COMP30830_Project

## Common parameters
> **[TEMPORARY SOLUTION]** Set the following parameters as a [.env](https://pypi.org/project/python-dotenv/) file when running locally. 

> **[IMPORTANT]** Include the .env file in **.gitignore** when pushing your commit.

### Database
- `DB_USER`: Your own database user name
- `DB_PASSWORD`: Your own database user password
- `DB_PORT`: Your database connection port (e.g. 3306)
- `DB_NAME`: Your database name (e.g. local_databasejcdecaux)
- `DB_URI`: Your database address (e.g. 127.0.0.1)

### OpenWeather
- `OWKEY`: This is your OpenWeather API key
- `CURRENT_URI `: [OpenWeather current weather data](https://openweathermap.org/current?collection=current_forecast)
